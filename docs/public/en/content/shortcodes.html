Shortcodes en fr

Shortcodes are small things that you can use in your contents.

Internal contents

  • Reference: use {{ ref("path/to/content.md") }} to generate a link to this content.
    • ⚠️ if you create a loop, beulogue will crash.
    • beulogue will give you a warning if you reference a non existing content.

External contents

  • Dailymotion: use {{ dailymotion("12345") }} to embed a Dailymotion video.
  • GitHub Gist: use {{ gist("beulogue", "123456") }} to embed a Gist.
  • vimeo: use {{ vimeo("12345") }} to embed a Vimeo video.
  • YouTube: use {{ youtube("12345") }} to embed a YouTube video.
